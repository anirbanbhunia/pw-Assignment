i) An HTML element is defined by a starting tag. If the element contains other content, it ends with a closing tag. For example, <p> is starting tag of a paragraph and </p> is closing tag of the same paragraph but <p>This is paragraph</p> is a paragraph element.

4.Brifly explain what HTML entities are.
HTML entities are special characters that are used to display reserved HTML characters, such as < and >, within HTML documents. These characters are represented by a unique code, such as < and >, which the browser will interpret as the corresponding special character. This allows developers to use reserved characters within HTML without causing the browser to interpret them as markup. Some examples of HTML entities include:
< for <
> for >
& for &
© for ©
