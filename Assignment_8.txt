2.Explain all media elements and how they work in html.

In html there are three media element audio,video,image.
for audio we use <audio></audio> tag
It is an inline element that is used to embed sound files into a web page.

for video we use <video></video> tag
It is used to embed video files into a web page.

for image we use <img> tag.
The <img> tag is used to embed an image in an HTML page.

3.What is get and post in form?

Get:GET is used to request data from a specified resource.
Note that the query string (name/value pairs) is sent in the URL of a GET request.

post:POST is used to send data to a server to create/update a resource.
The data sent to the server with POST is stored in the request body of the HTTP request.
